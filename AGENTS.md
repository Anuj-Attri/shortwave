# Shortwave – Agent Instructions

This repository is an experimental AI-assisted project.

The goal is to build **Shortwave**, a swipe-based music discovery app.

## Development rules

- Work through **Pull Requests**.
- Create a new branch for each task.
- Keep commits small and focused.
- Do not modify files unrelated to the task.

## Tech stack

Frontend
- React
- Vite
- TailwindCSS
- Framer Motion

Backend (later)
- Node / Express

## Commands

Install dependencies

npm install

Run dev server

npm run dev

Build

npm run build

## Project goal

Build a mobile-first swipe interface that lets users discover music through short previews and AI recommendations.

Focus on:
- clean UI
- modular components
- maintainable structure
